HighRoll Overlay Test

Instructions:

This is a test application for the HighRoll overlay window. Use the following keyboard shortcuts:

  • Press Alt+Shift+O to toggle the overlay visibility
  • Press Alt+Shift+C to toggle click-through mode

When click-through is enabled, you can click through the overlay window to interact with windows behind it.

Try opening other applications and test if the overlay stays on top.

Try dragging the overlay window by its header.
